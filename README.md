# Auth Kit - Server

# Endpoints
| **Method** | **Path**    | **Purpose**            |
|------------|-------------|------------------------|
| POST       | /signup     | Create new user        |
| POST       | /login      | Login and receive jwt  |
| POST       | /refresh    | Refresh a jwt          |
| GET        | /me         | Get current user info  |
| POST       | /org/create | Create an Organization |
